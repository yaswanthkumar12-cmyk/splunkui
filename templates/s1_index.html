Create a New Splunk App


Ensure your file includes the following columns:

  • config_type: The type of configuration file (e.g., `inputs`, `outputs`, `server`)
  • stanza_type: The type of stanza (e.g., `monitor`, `tcpout`)
  • stanza_value: The value associated with the stanza (e.g., `/var/log/*.log`, or leave it empty if not applicable)
  • key: The configuration parameter name (e.g., `host`, `index`, etc.)
  • value: The value for the configuration parameter (this can be a single value or comma-separated if it allows multiple values, like `source1, source2`)
